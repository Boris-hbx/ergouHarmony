对话主页

核心页面
Android 版 · 对话主页

布局结构

顶栏左: ≡ 汉堡菜单 + "二狗" 标题；右: 🕐历史 + ⚙设置
顶栏组件自定义 Row，高度 52vp，白色背景，无底边框
消息区List 组件，白色背景，initialIndex 末尾
快捷栏输入框上方，Scroll 横向，scrollBar(BarState.Off)
输入栏Row：TextInput + 发送 Button
无底部Tab所有功能通过 ≡ 抽屉菜单进入

消息气泡

用户气泡右对齐，#DCE8FF 淡蓝，黑字，圆角 16vp（右下 4vp）
助手气泡左对齐，#F2F3F5 浅灰，黑字，圆角 16vp（左下 4vp）
字号15fp，行高 1.7
消息操作assistant 下方：📋复制 👍点赞 👎点踩 🔄重新生成，16vp 图标
流式显示逐字追加，末尾闪烁 | 光标
Markdown支持加粗、列表、代码块、引用

快捷栏 Chip

圆角18vp
字号14fp
间距横向 8vp
边框1px #DDD
首期可用"记账" 正常色；任务/例行/出差/英语/养生 灰色 disabled
点击行为记账 → 跳转记账页（或输入框填充引导语）

输入栏

输入框TextInput，圆角 20vp，#F1F3F5 背景
高度最小 40vp，最大 100vp，多行自适应
发送按钮40×40vp 圆形。有内容: 蓝色 ▶；空: 灰色 disabled
停止按钮流式生成中 → 按钮变红色 ■（停止）

Android 版 · 记一笔弹窗

弹窗结构

弹出方式bindSheet 底部半屏，圆角 24vp

编辑账单

Android 版 · 编辑账单（含明细+票据）

与"记一笔"的区别

标题"编辑账单"（非"记一笔"）
字段预填所有字段从已有记录填充
明细区域显示 AI 分析结果：商品项 + 税费 + 小计 + Tip + 总计
英文翻译明细中可点击"英文翻译"查看原文
标签标注原始 表示 AI 提取的原始分类
Tip 高亮小费行用 黄色底色标注
总计橙色底色，加粗金额
描述AI 生成摘要（"在安大略省...餐厅用餐"），可展开查看全文
票据照片已有照片缩略图（可 ✕ 删除）+ 新增按钮
底部按钮删除（红字）+ 取消 + 保存（蓝色）
首期简化：明细解析（AI 识别票据行项目）是高级功能，首期编辑弹窗与"记一笔"共用同一组件， 仅增加预填数据 + 删除按钮。明细解析待照片功能完善后迭代。

功能广场

鸿蒙版改为抽屉菜单
Android 版 · 功能广场（2×3 宫格）

Android → 鸿蒙 变更说明

Android 版的"功能广场"是独立页面，2×3 宫格展示 6 个功能模块。

鸿蒙版不做独立功能广场页面，改为 ≡ 侧边抽屉菜单（见 #2）。
理由：首期只有对话+记账，宫格太空旷；抽屉更适合逐步扩展。
